## Description
Vue.jsとSassの学習を兼ねて簡単な電卓アプリを作りました。

- Vue.js
- Sass

## Feature
- 桁が増えすぎたときは、フォントサイズを小さくすることで画面に収まるようしました。
- あえてEnterキーは作っておらず入力に応じて随時計算結果を出力するようにしてます。
- いきなり 0 を入力できないようにしました。
